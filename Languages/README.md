所有编程语言和二进制文件格式笔记

编程语言：
- [[C++]]：剩余模板部分
- [[Python]]：剩余 tkinter 部分
- [[SQL]]：Structured Query Language，结构化查询语言
- [[Scala]]：JVM 平台带有面向函数特点的面向对象语言，但 Java 的不断更新也在丰富函数式的特征
- [[Groovy]]：JVM 平台的动态脚本语言，语法灵活但效率较低
- [[CSharp]]：C#，微软 .Net 平台语言
- [[Rust]]：结构体+Trait实现对象，无指针无 GC 语言，利用严格的生命周期避免了悬垂指针的产生

二进制文件格式：
- [[Class 字节码]]：JVM 平台语言编译的最终产物
